The Globe

Home

Partly inspired by the GitHub globe, and partly by the rebel briefing in Return of the Jedi, I wanted to see if I could learn enough WebGL to draw a globe made of points. This started with reading through webglfundamentals.org but then moved on to using three.js.

The colours for each point are based on the Blue Marble image from NASA; We read in a 3840x1920 version of the image then map each point on the globe to a point in the image and get its colour.
